# Getting Started
* Download cache and extract it to /data/world/cachers2cache if rs2cache folder is not there make sure to make it.
* First to start clone the project and open it up into eclipse make sure u are using JRE-1.8 for the project.
* Run "Launcher" Excute the Link Server using this argument: -link
* Then we want to start up the game world do the same process by running "Launcher" again and execute this argument: -game 1 dual-proto
* Make sure to setup database in data/world/config.ini
